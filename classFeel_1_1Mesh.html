Detailed Description

template<typename GeoShape, typename T = double, int Tag = 0, typename IndexT = uint32_type>
class Feel::Mesh< GeoShape, T, Tag, IndexT >

unifying mesh class

This structure is an aggregation of elements, faces, edges(3D) and points data structures and provides a unified interface with respect to the dimension.

Template Parameters
GeoShapeGeometric entities type
Tnumerical type for coordinates
Tag= 0 to discriminate between meshes of the same type

Finds all the processors that may contain elements that neighbor my elements. This list is guaranteed to include all processors that border any of my elements, but may include additional ones as well. This method computes bounding boxes for the elements on each processor and checks for overlaps.
