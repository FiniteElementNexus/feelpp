Feel::FaceDof< SizeT > Struct Template Reference

Description

template<typename SizeT = uint32_type>
struct Feel::FaceDof< SizeT >

Describe a Dof on a Face.

the data structure is a tuple containing

  • global dof id
  • sign of the dof
  • is dof periodic
  • local dof id in element
  • local dof id in face
